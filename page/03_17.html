canReadGameFiles () Boolean [static]

檢測瀏覽器是否可以讀取遊戲目錄中的文件。

返回值：

Boolean:
若瀏覽器可以讀取遊戲目錄中的文件，返回 true

isAndroidChrome () Boolean [static]

檢測瀏覽器是否為安卓版的 Chrome 瀏覽器。

返回值：

Boolean:
若瀏覽器為安卓版的 Chrome 瀏覽器，返回 true

isMobileDevice () Boolean [static]

檢測遊戲平台是否為移動設備。

返回值：

Boolean:
若遊戲平台為移動設備，返回 true

isMobileSafari () Boolean [static]

檢測瀏覽器是否為移動版的 Safari 瀏覽器。

返回值：

Boolean:
若瀏覽器為移動版的 Safari 瀏覽器，返回 true

isNwjs () Boolean [static]

檢測遊戲平台是否為 NW.js。

返回值：

Boolean:
若遊戲平台為 NW.js，返回 true

isOptionValid ( name ) Boolean [static]

檢測選項是否在查詢的字符串中。

參數：

  • name String

    選項名稱

返回值：

Boolean:
若選項在查詢的字符串中，返回 true

rgbToCssColor  ( r , g , b ) String [static]

將 RGB 值轉換成 CSS 顏色字符串。

參數：

  • r Number

    紅色通道強度（0～255）

  • g Number

    綠色通道強度（0～255）

  • b Number

    藍色通道強度（0～255）

返回值

String:
CSS 顏色字符串
