方法

addLoadListener ( listner )

Add a callback function that will be called when the audio data is loaded.

參數：

  • listner Function

    回調函數

addStopListener ( listner )

Add a callback function that will be called when the playback is stopped.

參數：

  • listner Function

    回調函數

canPlayM4a () Boolean [static]

檢測瀏覽器能否播放 m4a 文件。

Returns:

Boolean:
若瀏覽器支持播放 m4a 文件，返回 True 

canPlayOgg () Boolean [static]

檢測瀏覽器能否播放 ogg 文件。

Returns:

Boolean:
若瀏覽器支持播放 ogg 文件，返回 True 

clear ()

清除音頻數據。

fadeIn ( duration )

淡入音頻。

參數：

  • duration Number

    淡入的時間（秒）

fadeOut ( duration )

淡出音頻

參數：

  • duration Number

    淡出的時間（秒）

initialize ( noAudio ) Boolean [static]

初始化音頻系統。

參數：

  • noAudio Boolean

    無聲模式的標誌

返回值：

Boolean:
若音頻系統能夠正常使用，返回 True

isError () Boolean

檢測加載時是否有錯誤發生。

Returns:

Boolean:
若發生錯誤，返回 True 

isPlaying () Boolean

檢測音頻是否正在播放。

返回值：

Boolean:
若正在播放，返回 True 

isReady () Boolean

檢測音頻是否已經可以開始播放。

返回值：

Boolean:
若音頻可以開始播放，返回 True 

play ( loop , offset )

播放音頻。

參數：

  • loop Boolean

    是否要循環播放

  • offset Number

    播放開始的時間（秒）

seek ()

獲取當前音頻的播放位置。

stop ()

停止播放音頻。

屬性

pan Number

音頻的聲場位置。

pitch Number

音頻的音調。

url String

[只讀] 音頻文件的地址。

volume Number

音頻的音量。
