Change HP（更改 HP）

功能

更改角色的 HP。

設置

Actor（角色）
選擇目標角色。選擇[Fixed（固定）]選項時，你需要指定特定的角色（或者是全體同伴）。當你需要用角色 ID 來指定角色時，請選擇[Variable（變量）]選項，然後指定要引用的變量。
Operation（操作）
指定是增加 HP 還是減少 HP。
Operand（操作數）
指定增加或減少 HP 的數量。選擇[Constant（常量）]來設定一個固定值，你需要輸入 1 ~ 9999 之間的數值。當使用變量設置增減數量時，選擇[Variable（變量）]，然後指定要引用的變量。
Allow Knockout（允許死亡）
啟用時，角色的 HP 低於 0 後他們會死亡。禁用時，若角色的 HP 在 0 以下，那麼 HP 將會強制設置為 1。

Change Skill（更改技能）

功能

控制角色所能使用的技能。

設置

Actor（角色）
選擇目標角色。選擇[Fixed（固定）]選項時，你需要指定特定的角色。當你需要用角色 ID 來指定角色時，請選擇[Variable（變量）]選項，然後指定要引用的變量。

Change Equipment（變更裝備）

功能

更改角色的裝備。

設置

Actor（角色）
選擇目標角色。
Equipment Type（裝備類型）
指定要變更的裝備類型。
Equipment Item（裝備物品）
指定要換上的裝備。隊伍中必須要有這件裝備。

注記

  • 如果隊伍中沒有這個裝備，角色的裝備不會變化。要強制裝備一項物品，你必須要先為隊伍增加相應的裝備，這利用[Change Weapons（增減武器）] 和 [Change Armors（增減防具）] 事件指令可以做到。

Change Class（更改職業）

功能

更改角色的職業。

設置

Actor（角色）
選擇目標角色。
Class（職業）
指定變更後的新職業。
Save Level（保留等級） （2016.03.01 更新）
此項被選中時，職業改變後角色的等級不會發生變化。

注記

  • 更改職業後，該職業不能裝備的武器和防具會被自動卸下。
