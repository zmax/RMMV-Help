addChild ( child ) Object

參數：

  • child Object

    要添加的子對象。

返回值：

Object:
被添加的子對象。

addChildAt ( child  index ) Object

將子對像添加到指定索引位置。

參數：

  • child Object

    要添加的子對象。

  • index Number

    欲添加到的位置索引。

返回值：

Object:
被添加的子對象。

addChildToBack ( child ) Object

添加一個子對像於內容和背景之間。

參數：

  • child Object

    要添加的子對象。

返回值：

Object:
被添加的子對象。

isClosed ()

如果窗口被成功關閉（窗口的開啟程度為0），則返回 true 。

isOpen ()

如果窗口被成功打開（窗口的開啟程度為255），則返回 true 。

move ( x , y , width , height )

同時設置x坐標、y坐標、寬度和高度。

參數：

  • x Number

    窗口的x坐標。

  • y Number

    窗口的y坐標。

  • width Number

    窗口的寬度。

  • height Number

    窗口的高度。

removeChild ( child ) Object

從容器中移除一個子對象。

參數：

  • child Object

    要移除的子對象。

返回值：

Object:
被移除的子對象。

removeChildAt ( index ) Object

移除指定索引處的子對象。

參數：

  • index Number

    要移除的子對象的索引。

返回值：

Object:
被移除的子對象。

setCursorRect ( x , y , width , height )

設置光標存放區域。

參數：

  • x Number

    光標的x坐標。

  • y Number

    光標的y坐標。

  • width Number

    光標的寬度。

  • height Number

    光標的高度。

setTone ( r , g , b )

改變背景的色調。

參數：

  • r Number

    介於(-255, 255)之間的紅色值。

  • g Number

    介於(-255, 255)之間的綠色值。

  • b Number

    介於(-255, 255)之間的藍色值

返回值：


update ()

更新窗口。

屬性

active Boolean

光標閃爍的狀態。

backOpacity Number

窗口背景的不透明度（0~255）。

children Array

[只讀]窗口的子對像數組。

contents Bitmap

窗口內容的引用位圖。

contentsOpacity Number

窗口內容的不透明度（0~255）。

downArrowVisible  Number

下滾箭頭的可見性。

height Number

窗口的高度。

margin Number

窗口的外邊距。

opacity Number

窗口的不透明度（0~255）。

openness Number

窗口的開啟程度（0~255）。

origin Point

窗口滾動的初始點。

padding Number

窗口框架與內容之間的填充寬度（內邊距）。

parent Object

[只讀]窗口的父對象。

pause Boolean

暫停標記的可見性。

upArrowVisible Boolean

上滾箭頭的可見性。

visible Boolean

窗口的可見性。

width Number

窗口的寬度。

windowskin Bitmap

窗口外觀的引用位圖。

x Number

窗口的x坐標。

y Number

窗口的y坐標。
