以平鋪方式顯示圖片的 sprite（精靈） 對象。

構造函數

TilingSprite ( bitmap )

參數：

  • bitmap Bitmap

    需要平鋪顯示的圖片

方法

move ( x , y , width , height )

直接設置 TilingSprite（平鋪精靈） 的 X 坐標、Y 坐標、寬度和高度。

參數：

  • x Number

    TilingSprite（平鋪精靈） 的 X 坐標

  • y Number

    TilingSprite（平鋪精靈） 的 Y 坐標

  • width Number

    TilingSprite（平鋪精靈） 的寬度

  • height Number

    TilingSprite（平鋪精靈） 的高度

setFrame ( x , y , width , height )

設置 TilingSprite（平鋪精靈） 所顯示 bitmap（位圖） 的矩形區域。

參數：

  • x Number

    矩形區域的 X 坐標

  • y Number

    矩形區域的 Y 坐標

  • width Number

    矩形區域的寬度

  • height Number

    矩形區域的高度

update ()

在每一幀中刷新 TilingSprite（平鋪精靈） 。

屬性

bitmap Bitmap

TilingSprite（平鋪精靈） 所顯示的圖片。

opacity Number

TilingSprite（平鋪精靈） 的不透明度（0 ~ 255）。

origin Point

滾動顯示效果中 TilingSprite（平鋪精靈） 的原點。

visible Boolean

TilingSprite（平鋪精靈） 的可見性。

x Number

TilingSprite（平鋪精靈） 的 X 坐標。

y Number

TilingSprite（平鋪精靈） 的 Y 坐標。
