方法

addChild ( child ) Object

向容器中添加子元素。

參數：

  • child Object

    待添加的子元素

返回值：

Object:
被添加的子元素

addChildAt ( child  index ) Object

在容器指定索引的位置添加子元素。

參數：

  • child Object

    待添加的子元素

  • index Number

    指定的索引

返回值：

Object:
被添加的子元素

isReady () Boolean

檢測圖塊組是否可以開始渲染。

返回值：

Boolean:
若可以開始渲染，返回 true 

refresh ()

強制重繪整個 tilemap（圖塊地圖）。

removeChild ( child ) Object

從容器中移除子元素。

參數：

  • child Object

    待移除的子元素

返回值：

Object:
被移除的子元素

removeChildAt ( index ) Object

移除容器指定索引位置的子元素。

參數：

  • index Number

    指定的索引

返回值：

Object:
被移除的子元素

setData ( width  height  data )

設置 tilemap（圖塊地圖） 的數據。

參數：

  • width Number

    地圖的橫向圖塊數

  • height Number

    地圖的縱向圖塊數

  • data Array

    地圖數據的一維數組

update ()

在每一幀中刷新 tilemap（圖塊地圖）。

屬性

animationCount Number

自動圖塊的動畫計數器。

bitmaps Array

地圖圖塊的 Bitmap（位圖） 對像數組。

children Array

[只讀] tilemap（圖塊地圖） 的子元素數組。

flags Array

圖塊組的標誌列表。

height Number

遊戲畫面的高度（像素）。

horizontalWrap Boolean

tilemap（圖塊地圖） 是否橫向循環。

origin Point

tilemap（圖塊地圖） 捲動的起始坐標。

parent Object

[只讀]包含此 tilemap（圖塊地圖） 的父容器。

tileHeight Number

單個圖塊的高度（像素）。

tileWidth Number

單個圖塊的寬度（像素）。

verticalWrap Boolean

tilemap（圖塊地圖） 是否縱向循環。

width Number

遊戲畫面的寬度。
