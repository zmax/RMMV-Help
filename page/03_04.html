方法

clear () [static]

清除全部的輸入數據。

initialize () [static]

初始化輸入系統。

isLongPressed  ( keyName ) Boolean [static]

檢測某個按鍵是否正被長時間按下。

參數：

  • keyName String

    待檢測按鍵的映射名

返回值：

Boolean:
若按鍵正被長時間按下，返回 true

isPressed ( keyName ) Boolean [static]

檢測按鍵是否正被按下。

參數：

  • keyName String

    待檢測按鍵的映射名

返回值：

Boolean:
若按鍵正被按下，返回 true

isRepeated ( keyName ) Boolean [static]

檢測按鍵是否正被重複按下。

參數：

  • keyName String

    待檢測按鍵的映射名

返回值：

Boolean:
若按鍵正被重複按下，返回 true

isTriggered ( keyName ) Boolean [static]

檢測按鍵是否剛被按下。

參數：

  • keyName String

    待檢測按鍵的映射名

返回值：

Boolean:
若按鍵剛被按下，返回 true

update () [static]

更新輸入數據。

屬性

date Number [static]

[只讀] 距離上一次按鍵所經過的時間（毫秒）。

dir4 Number [static]

[只讀] 小鍵盤上與四個方向相對應的數字，無方向則為0。

dir8 Number [static]

[只讀] 小鍵盤上與八個方向對應的數字，無方向則為0。

gamepadMapper Object [static]

將手柄按鍵轉換為按鍵映射名的散列表。

keyMapper Object [static]

將虛擬按鍵轉換為按鍵映射名的散列表。

keyRepeatInterval Number [static]

更新重複按鍵狀態的間隔時間（幀）。

keyRepeatWait Number [static]

進入長按狀態前的等待時間（幀）。
