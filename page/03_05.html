Array.prototype.contains ( element ) Boolean

檢測數組中是否包含指定元素。

參數：

  • element Any

    待檢測的元素

返回值：

Boolean:
若數組中包含指定元素，返回 true

Array.prototype.equals ( array ) Boolean

參數：

  • array Any

    需要進行比較的數組

返回值：

Boolean:
若兩個數組相等，返回 true

Math.randomInt ( max ) Number [static]

隨機生成一個介於 0 到 max-1 之間的整數。

參數：

  • max Number

    隨機數的最大值（生成的隨機數不包含該值）

返回值：

Number:
隨機的整數

Number.prototype.clamp ( min , max ) Number

返回一個大小不超出指定範圍的數字值。

參數：

  • min Number

    指定範圍的下限

  • max Number

    指定範圍的上限

返回值：

Number:
大小不超出指定範圍的值

Number.prototype.mod ( n ) Number

Number:
