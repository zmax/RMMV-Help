在RPG Maker MV中，你的新工程中已經有了一些插件，你可以利用簡單的設置來使用他們。

AltMenuScreen

一個將菜單畫面修改為另一個佈局的插件。
隊伍成員會水平地顯示。

AltSaveScreen

一個將存檔畫面和讀檔畫面修改為另一種佈局的插件。
可以讓顯示圖片成為可能，例如顯示隊伍成員的頭像。

EnemyBook

怪物敵人圖鑒

你可以製作一本《怪物敵人圖鑒》，其中列出玩家擊敗的敵人。

插件指令

在事件指令的[插件指令]中可以輸入並執行以下內容。

EnemyBook open 打開敵人圖鑒畫面
EnemyBook add 3 添加3號敵人到敵人圖鑒
EnemyBook remove 4 從敵人圖鑒中移除4號敵人
EnemyBook complete 完成敵人圖鑒
EnemyBook clear 清除敵人圖鑒

敵人註釋

在數據庫的「敵人」標籤頁中，修改「註釋」區域。

<desc1:description 1> 第一行顯示的描述
<desc2:description 2> 第二行顯示的描述
<book:no> 在不打算放入敵人圖鑒時候使用

ItemBook

道具圖鑒

你可以製作一本《道具圖鑒》，其中列出玩家收集的所有裝備和物品。

插件指令

在事件指令的[插件指令]中可以輸入並執行以下內容。

ItemBook open 打開道具圖鑒畫面
ItemBook add weapon 3 在道具圖鑒中增加3號武器
ItemBook add armor 4 在道具圖鑒中增加 4 號防具
ItemBook remove armor 5 在道具圖鑒中移除 5 號防具
ItemBook remove item 6 在道具圖鑒中移除 6 號物品
ItemBook complete 完成道具圖鑒
ItemBook clear 清除道具圖鑒

物品（武器、防具）註釋

在數據庫的「物品（武器、防具）」標籤頁中，修改「註釋」區域。

<book:no> 在不打算放入道具圖鑒時候使用

SimpleMsgSideView

隱藏橫板戰鬥中顯示的戰鬥日誌。只有技能、物品和動作（攻擊以及防禦）的名字會被顯示在屏幕的上方。

參數

displayAttack
是否顯示一次普通的「攻擊」。
設置為0（默認）時不顯示，設置為1的時候顯示。
position
在左邊或者中間顯示技能和物品的名字。
設置為0的時候在左邊顯示，設置為1的時候在中間顯示。

TitleCommandPosition

一個修改標題畫面中的命令窗口的位置和背景的插件。

參數

Offset X
X坐標的偏移值。
Offset Y
Y坐標的偏移值。
Width
命令窗口的寬度。
Background
背景類型。你可以選擇三種不同的類型：0 - 普通， 1 - 暗， 2 - 透明。

WeaponSkill

一個修改每個武器的普通攻擊對應的技能ID的插件。

創建一個你想附加在武器上的技能。在武器「註釋」區域指定技能ID，就能在使用這個武器攻擊時激活那個技能。
例子： <skill_id:3>

你可以創建以下種類的武器。

  • 創建攻擊多個敵人的武器。
  • 創建攻擊2或者3次的武器。
  • 如果技能被設置為回復魔法，當你選擇「攻擊」時，可以選擇一個隊友並使他們回復。
  • 可以製作一個相當於防禦指令的武器，等。

將使用的武器所設置的技能的「技能類型」設置為「無」。
這樣做防止技能類型被禁止使用的情況下，由於技能被禁止，玩家無法選擇「攻擊」。

觸屏UI （2016 年 3 月 1 日更新）

該插件可以將當前 UI（遊戲界面） 轉換為觸屏操作。
* 本插件不能和其他修改了 UI 的插件共同使用。
（例如 AltMenuScreen、AltSaveScreen、SimpleMsgSideView、TitleCommandPosition 等）

參數

Menu Command Text
在地圖上，顯示在打開菜單按鈕上的文字。
Previous Command Text
在菜單中，顯示在上一個按鈕上的文字。
Next Command Text
在菜單中，顯示在下一個按鈕上的文字。

扭蛋 / 扭蛋圖鑒 （2016 年 3 月 1 日更新）

使用本插件可以添加扭蛋系統。

扭蛋

一個可以隨機獲得物品的系統。

參數

Help Message Text
顯示在扭蛋界面的幫助文字。Required Amount is replaced by required G.
Button Text
扭蛋按鈕上的文字。
Get Message Text
扭蛋結束後顯示的文字。Item Name will be replaced by the obtained Item Name.
Show Item Description
打開/關閉 獲得扭蛋後顯示扭蛋說明的功能。
設置成 0 則不顯示說明，設置成 1 則顯示說明。
Effect
設置獲得扭蛋時播放的動畫編號。
Rank1～5 Effect
設置獲得 星級1 到 星級5 品質的扭蛋時各自所播放的動畫編號。
ME
設置獲得扭蛋時播放的 ME 。
Required Amount
設置每次扭蛋所需消耗的金錢。

插件指令

在事件指令的[插件指令]中可以輸入並執行以下內容。

Gacha open 打開扭蛋界面。
Gacha add item 1 向扭蛋池中添加 1 號物品。
Gacha remove item 1 從扭蛋池中移除 1 號物品。
Gacha clear 清空扭蛋池。

物品（武器、防具）註釋

在數據庫的「物品（武器、防具）」標籤頁中，修改「註釋」區域。

<gachaImage: image> 設置扭蛋的圖片。圖片文件放在 img/gacha/ 目錄下。
<gachaNumLot: 10> Sets number of Gacha.
<gachaRank: 5> 設置扭蛋的星級（1 到 5）。

扭蛋圖鑒。

顯示獲得的扭蛋列表。

參數

Unknown Data
未收集扭蛋的名字。
Price Text
This is the text for [Price].
Equip Text
This is the text for [Equipment].
Type Text
This is the text for [Type].
Rank Text
This is the text for [Rank].
Simple Display
切換扭蛋說明的顯示方式。設置成 0 則以普通方式顯示，設置成 1 則僅顯示簡單信息（名稱和說明）。

插件指令

在事件指令的[插件指令]中可以輸入並執行以下內容。

GachaBook open 打開扭蛋圖鑒。
GachaBook add weapon 3 將 3 號武器添加到扭蛋圖鑒中。
GachaBook add armor 4 將 4 號防具添加到扭蛋圖鑒中。
GachaBook remove armor 5 將 5 號防具移除出扭蛋圖鑒。
GachaBook remove item 6 將 6 號物品移除出扭蛋圖鑒。
GachaBook clear 清空扭蛋圖鑒。

老虎機 （2016 年 3 月 1 日更新）

使用本插件可以添加老虎機系統。

參數

Variable ID
This is this variable ID to manage the amount of coins owned.
Help Text
幫助文字。
Won Text
This is the message shown when winning. Win Coin will be replaced by the amount of obtained coins.
Lost Text
失敗時顯示的文字。
Replay Text
This is the message shown when choosing to retry or not.
Coin Full Text
This is the message shown when coins reach maximum amount.
Bet Text
加注指令的文字。
Spin Text
拉下拉桿指令的文字。
Yes Text
確定指令的文字。
No Text
取消指令的文字

插件指令

在事件指令的[插件指令]中可以輸入並執行以下內容。

SlotMachine open 打開老虎機。
SlotMachine expectation 0.5 Sets expected value to a number in between 0 to 1. The closer to the value 1, the better chance of winning, though there is never a 100% chance.
SlotMachine scale 0 設置倍率。值為 0 則是 x1，值為 1 則是 x10，值為 2 則是 x100 。
