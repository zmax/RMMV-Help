構建遊戲發行包

您可以為遊戲構建包括 Windows、Mac OS X、安卓和 iOS 在內全部平台的發行包。

Windows平台
選擇打包後將導出遊戲的工程文件夾。
如果您希望將文件夾打包成單獨的 .exe 文件，請參考 [使用 Enigma Virtual Box] 部分。
Mac OS X
請發佈遊戲的工程文件夾以及文件夾下的 Game.app 文件。
* 在 Windows 平台下 Game.app 會顯示成了一個文件夾，請不用擔心，它可以在 Mac OS X 下正常運行。
安卓/iOS
請參考 [轉換為安卓應用] 和 [轉換為 iOS 應用] 部分。
網頁瀏覽器（2016 年 3 月 1 日更新）
將文件上傳到網絡服務器上後，您可以直接在網頁瀏覽器上運行遊戲。

運行遊戲所需的系統要求
系統要求 需要支持 HTML5 Canvas 以及 Web Audio API
iOS 8 或更高版本
Android 4.4 或更高版本
推薦運行環境 Windows, Android: Google Chrome 瀏覽器
Mac OS X, iOS: Safari 瀏覽器
* 在線進行遊戲時請使用 Wi-Fi 。
* 推薦使用支持 WebGL 的瀏覽器進行遊戲。

選項 [Exclude unused files（不包含未使用文件）] （2016 年 3 月 1 日更新）

勾選該選項，則遊戲發佈時將不包含未使用的文件。使用這一功能，您可以縮減遊戲發行包的體積。

以下文件將被排除。

  • 工程目錄/img/（該目錄中的未使用文件）
  • 工程目錄/audio/（該目錄中的未使用文件）

請參考 [[不包含未使用文件] 插件配置]部分來查看如何讓本功能與插件相適應。
