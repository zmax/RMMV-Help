因為安全的原因，無法直接在瀏覽器中打開工程目錄下的index.html文件。
你必須在你的電腦中搭建一個本地服務器。本小節會解釋如何做。

在Windows下搭建本地服務器

開啟Internet Information Service

  1. 在「控制面板」中，點擊「程序」。
  2. 點擊「啟用或關閉Windows功能」。
  3. 在「Windows功能」窗口中，點擊「Internet Information Services」來啟用它。
    *不是選中，是不完全選中
  4. 點擊「確定」。

文件的安裝位置

Internet Information Services 被啟用時，會自動創建一個本地服務器的文件夾。把工程文件夾放在這個文件夾中。

  1. 點擊「此電腦」 → 「本地磁盤 (C:)」。
  2. 打開「inetpub」文件夾中的「wwwroot」文件夾。
  3. 把你的工程文件夾放在「wwwroot」文件夾中。

在你的瀏覽器中訪問

  1. 打開你的瀏覽器。
  2. 在地址欄中輸入以下地址並確認。
    http://<IP地址>/工程文件夾名稱/
    或
    http://<計算機名>/工程文件夾名稱/
    或
    http://localhost/工程文件夾名稱/

    例) http://192.168.0.1/工程文件夾名稱/

尋找計算機名

  1. 打開文件資源管理器窗口，在側邊欄中右鍵點擊「此電腦」並點擊「屬性」。
  2. 在「系統」窗口的「計算機名、域和工作組設置」區域下方的「計算機名：」右邊找到你的計算機的名稱。

找到你的IP地址

  1. 在「控制面板」中點擊「查看網絡狀態和任務」。
  2. 在網絡和共享中心窗口中，點擊「連接：本地連接」。
  3. 在出現的本地連接狀態窗口中，點擊「詳細信息...」。
  4. 在「IPv4 地址」右邊找到的數字就是你的IP地址。

設置localhost

要在瀏覽器中訪問 http://localhost/ ，你需要設置「hosts」文件。

  1. 點擊「開始」按鈕，進入「所有程序」 → 「附件」。右鍵點擊「記事本」並點擊「以管理員身份運行」。
  2. 在記事本中點擊「文件」 → 「打開」，定位到 「C:\Windows\System32\drivers\etc」。選擇「所有文件 (*.*)並打開「hosts」文件。
  3. 在文件中修改以下位置，並保存文件。
    ■修改前 
    #   127.0.0.1       localhost
    ■修改後 
    127.0.0.1       localhost
  4. 重新啟動Windows。

Mac OSX 本地服務器搭建

在Mac電腦中，Apache是默認安裝的。你可以通過運行Apache來使用本地服務器。

Apache 基本操作

在終端中輸入命令，運行Apache。

運行 Apache

$ sudo apachectl start

退出 Apache

$ sudo apachectl stop

文件的安裝位置

把工程文件夾放到以下位置中。

/Library/WebServer/Documents

* 在建立新工程和保存時，工程文件夾會被定位在「位置」指定的位置中。

在你的瀏覽器中訪問

  1. 打開你的瀏覽器。
  2. 在地址欄中輸入以下地址並確認。
    http://localhost/工程文件夾名稱/
