構造函數

參數：

    位圖的寬度

  • height Number

    位圖的高度

方法

addLoadListener ( listner )

添加一個回調函數，在位圖被加載時調用該函數。

參數：

  • listner Function

    回調函數

adjustTone ( r , g , b )

改變位圖色調。

參數：

  • r Number

    紅色通道強度（-255～255）

  • g Number

    綠色通道強度（-255～255）

  • b Number

    藍色通道強度（-255～255）

blt ( source , sx , sy , sw , sh , dx , dy , [dw=sw] , [dh=sh] )

將源位圖給定矩形區域的數據傳送到當前位圖的給定矩形區域內。

參數：

  • source Bitmap

    源位圖

  • sx Number

    源位圖矩形區域左上角的 X 坐標

  • sy Number

    源位圖矩形區域左上角的 Y 坐標

  • sw Number

    源位圖矩形區域的寬度

  • sh Number

    源位圖矩形區域的高度

  • dx Number

    目標區域左上角的 X 坐標

  • dy Number

    目標區域左上角的 Y 坐標

  • [dw=sw] Number [可選參數]

    目標區域的寬度

  • [dh=sh] Number [可選參數]

    目標區域的高度

blur ()

對位圖執行模糊效果。

clear ()

清除位圖。

clearRect ( x , y , width , height )

清除位圖中指定矩形區域的內容。

參數：

  • x Number

    矩形區域左上角的 X 坐標

  • y Number

    矩形區域左上角的 Y 坐標

  • width Number

    矩形區域的寬度

  • height Number

    矩形區域的高度

drawCircle ( x , y , radius , color )

填充指定的圓形區域。

參數：

  • x Number

    圓形區域圓心的 X 坐標

  • y Number

    圓形區域圓心的 Y 坐標

  • radius Number

    圓形區域的半徑

  • color String

    圓形區域的顏色（CSS格式）

drawText ( text , x , y , maxWidth , lineHeight , align )

在位圖上繪製描邊文字。

參數：

  • text String

    要繪製的文字

  • x Number

    文字左側的 X 坐標

  • y Number

    文字頂部的 Y 坐標

  • maxWidth Number

    文字允許的最大寬度

  • lineHeight Number

    文字的行高

  • align String

    文字的對齊方式

fillAll ( color )

填充整個位圖

參數：

  • color String

    填充的顏色（CSS格式）

fillRect ( x , y , width , height , color )

填充指定的矩形區域。

參數：

  • x Number

    矩形區域的 X 坐標

  • y Number

    矩形區域的 Y 坐標

  • width Number

    矩形區域的寬度

  • height Number

    矩形區域的高度

  • color String

    填充的顏色（CSS格式）

getAlphaPixel ( x , y ) String

返回指定像素點的不透明度（alpha 值）。

Parameters:

  • x Number

    位圖中指定點的 X 坐標

  • y Number

    位圖中指定點的 Y 坐標

返回值：

String:
不透明度（alpha 值）

getPixel ( x , y ) String

返回指定像素點的顏色。

參數：

  • x Number

    位圖中指定點的 X 坐標

  • y Number

    位圖中指定點的 Y 坐標

返回值：

String:
顏色（十六進制格式）

gradientFillRect ( x , y , width , height , color1 , color2 , vertical )

以漸變模式填充指定的矩形區域。

參數：

  • x Number

    矩形區域的 X 坐標

  • y Number

    矩形區域的 Y 坐標

  • width Number

    矩形區域的寬度

  • height Number

    矩形區域的高度

  • color1 String

    漸變的起始顏色

  • color2 String

    漸變的結束顏色

  • vertical Boolean

    是否為縱向漸變

isError () Boolean

檢測位圖加載時是否發生錯誤。

返回值：

Boolean:
若發生錯誤，返回 true

isReady () Boolean

檢測位圖能否開始渲染。

返回值：

Boolean:
若位圖能夠開始渲染，返回 true

load ( url ) [static]

加載圖片文件，返回一個新的位圖對象。

參數：

  • url String

    加載圖片的地址

返回值：

Bitmap（位圖） 對像

measureTextWidth ( text ) Number

測量指定文字的寬度。

參數：

  • text String

    待測量的文字

返回值：

Number:
文字的寬度（像素）

resize ( width , height )

調整位圖大小。

參數：

  • width Number

    位圖調整後的寬度

  • height Number

    位圖調整後的高度

rotateHue ( offset )

變換位圖的色相。

參數：

  • offset Number

    色相環的偏移量（360度以內）

snap ( stage ) [static]

截取當前遊戲畫面，返回一個新的 Bitmap（位圖） 對象。

參數：

  • stage PIXI.Stage

    stage（場景） 對像

返回值：

Bitmap（位圖） 對像

屬性

[只讀] 位圖對象的 BaseTexture。

canvas HTMLCanvasElement

[只讀] 位圖對象的 Canvas。

context CanvasRenderingContext2D

[只讀] 位圖對象的 CanvasRenderingContext2D 對象。

fontFace String

字體名稱。

fontItalic Boolean

字體傾斜。

fontSize Number

字體大小。

height Number

[只讀] 位圖的高度。

outlineColor String

文字描邊的顏色（CSS格式）。

outlineWidth Number

文字描邊的厚度。

paintOpacity Number

位圖對象的不透明度（alpha 值）（0~255）。

rect Rectangle

[只讀] 位圖對象的矩形區域。

smooth Boolean

是否應用平滑縮放。

textColor String

文字的顏色（CSS格式）。

url String

[只讀] 圖片文件的地址。

width Number

[只讀] 位圖的寬度。
