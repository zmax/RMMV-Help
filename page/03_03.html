addLoadListener ( listner ) [static]

添加一個回調函數，當音頻文件被加載時調用該函數。

參數：

  • listner Function

    回調函數

clear () [static]

清除音頻數據。

fadeIn ( duration ) [static]

淡入音頻。

參數：

  • duration Number

    淡入的時間（秒）

fadeOut ( duration ) [static]

淡出音頻。

參數：

  • duration Number

    淡出的時間（秒）

initialize () Boolean [static]

初始化音頻系統。

返回值：

Boolean:
若音頻系統可以使用，返回 true

isError () Boolean [static]

檢測載入是否出錯。

返回值：

Boolean:
若載入時出現錯誤，返回 true

isPlaying () Boolean [static]

檢測音頻是否正在播放。

返回值：

Boolean:
若音頻正在播放，返回 true

isReady () Boolean [static]

檢測音頻數據是否已經可以開始播放。

返回值：

Boolean:
若可以開始播放，返回 true

play ( loop , offset ) [static]

播放音頻。

參數：

  • loop Boolean

    音頻是否要循環播放

  • offset Number

    音頻開始播放的位置（秒）

seek () [static]

獲取當前音頻播放的位置。

setup ( url ) [static]

設置 Html5 音頻。

參數：

  • url String

    音頻文件的地址

stop () [static]

停止播放音頻。

屬性

url String

[只讀] 音頻文件的地址。

volume Number

音頻的音量。
