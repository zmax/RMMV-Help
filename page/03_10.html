addChild ( child ) Object

向容器中添加子元素。

參數：

  • child Object

    待添加的子元素

返回值：

Object:
被添加的子元素

addChildAt ( child  index ) Object

在容器指定索引的位置添加子元素。

參數：

  • child Object

    待添加的子元素

  • index Number

    指定的索引

返回值：/h4>

Object:
被添加的子元素

getBlendColor () Array

獲取 sprite（精靈） 的混合顏色。

返回值：

Array:
混合顏色（[r, g, b, a]）

getColorTone () Array

獲取 sprite（精靈） 的色調。

返回值：

Array:
色調（[r, g, b, gray]）

move ( x , y )

直接設置 sprite（精靈） 的 X 坐標和 Y 坐標。

參數：

  • x Number

    sprite（精靈） 的 X 坐標

  • y Number

    sprite（精靈） 的 Y 坐標

removeChild ( child ) Object

從容器中移除子元素。

參數：

  • child Object

    待移除的子元素

返回值：

Object:
被移除的子元素

removeChildAt ( index ) Object

移除容器指定索引位置的子元素。

參數：

  • index Number

    指定的索引

返回值：

Object:
被移除的子元素

setBlendColor ( color )

設置 sprite（精靈） 的混合顏色。

參數：

  • color Array

    混合顏色（[r, g, b, a]）

setColorTone ( tone )

設置 sprite（精靈） 的色調。

參數：

  • tone Array

    色調（[r, g, b, gray]）

setFrame ( x , y , width , height )

設置 sprite（精靈） 所顯示 bitmap（位圖） 的矩形區域。

參數：

  • x Number

    矩形區域的 X 坐標

  • y Number

    矩形區域的 Y 坐標

  • width Number

    矩形區域的寬度

  • height Number

    矩形區域的高度

update ()

在每一幀中刷新 sprite（精靈）。

屬性

anchor Point

sprite（精靈） 的原點坐標（(0,0) ~ (1,1)）。

bitmap Bitmap

sprite（精靈） 所顯示的圖片。

blendMode Number

sprite（精靈） 所使用的混合模式。

children Array

[只讀] sprite（精靈） 的子元素數組。

filters Array

設置 sprite（精靈） 所使用的濾鏡。

height Number

sprite（精靈） 的高度（不考慮縮放）。

opacity Number

sprite（精靈） 的不透明度（0 ~ 255）。

parent Object

[只讀]包含此 sprite（精靈） 的父容器。

rotation Number

sprite（精靈） 的旋轉角度（弧度）。

scale Point

sprite（精靈） 的縮放比例。

visible Boolean

sprite（精靈） 的可見性。

width Number

sprite（精靈） 的寬度（不考慮縮放）。

x Number

sprite（精靈） 的 X 坐標。

y Number

sprite（精靈） 的 Y 坐標。
