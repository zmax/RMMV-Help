方法

clear () [static]

清除全部的輸入數據。

initialize () [static]

初始化輸入系統。

isCancelled () Boolean [static]

檢測鼠標右鍵是否剛剛被按下。

返回值：

Boolean:
若鼠標右鍵剛剛被按下，返回 true

isLongPressed () Boolean [static]

檢測鼠標左鍵或者觸屏是否長時間被按住。

返回值：

Boolean:
若鼠標左鍵或者觸屏長時間被按住，返回 true

isMoved () Boolean [static]

檢測鼠標或手指在觸屏上是否發生移動。

返回值：

Boolean:
若鼠標或手指在觸屏上發生移動，返回 true

isPressed () Boolean [static]

檢測鼠標或者觸屏是否正被按下。

返回值：

Boolean:
若鼠標或觸屏正被按下，返回 true

isReleased () Boolean [static]

檢測鼠標左鍵是否彈起或者手指是否離開觸屏。

返回值：

Boolean:
若鼠標左鍵彈起或者手指離開觸屏，返回 true

isRepeated () Boolean [static]

檢測鼠標左鍵或者觸屏是否剛剛被按下，或者是偽重複按鍵是否發生。

返回值：

Boolean:
若鼠標左鍵或者觸屏被重複按下，返回 true

isTriggered () Boolean [static]

檢測鼠標左鍵或者觸屏是否剛剛被按下。

返回值：

Boolean:
若鼠標左鍵或者觸屏剛剛被按下，返回 true

update () [static]

更新輸入數據。

屬性

date Number [static]

[只讀]距離上一次按鍵所經過的時間（毫秒）。

keyRepeatInterval Number [static]

更新重複按鍵狀態的間隔時間（幀）。

keyRepeatWait Number [static]

進入長按狀態前的等待時間（幀）。

wheelX Number [static]

[只讀]水平滾動量。

wheelY Number [static]

[只讀]垂直滾動量。

x Number [static]

[只讀]最近一次觸屏事件在 canvas 區域中的 X 坐標。

y Number [static]

[只讀]最近一次觸屏事件在 canvas 區域中的 Y 坐標。
