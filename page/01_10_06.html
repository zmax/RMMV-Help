Transfer Player（移動主角）

功能

改變主角隊伍的位置。

設置

Location（位置）
指定移動後的位置。如果想要將隊伍移動到指定位置，選擇[Direct designation（直接指定）]，然後點擊[...]按鈕，在打開的窗口中指定移動的位置。如果想要使用地圖 ID 和坐標指定位置，選擇[Designation with variables（使用變量指定）]，然後選擇要引用的變量的指定[ID]，[X 坐標]，[Y 坐標]。
Direction（方向）
指定角色移動後的朝向。
Fade（淡入淡出）
指定移動時畫面的過渡方式。當選擇了[Black（黑色）]或[White（白色）]時，畫面會相應地顯示為黑色和白色，然後主角就會出現在目標位置上。

Set Vehicle Location（設置交通工具的位置）

功能

改變交通工具的位置。

設置

Vehicle（交通工具）
指定目標交通工具（小型船/大型船/飛空艇）。
Location（位置）
指定移動後的位置。如果想要將交通工具移動到指定位置，選擇[Direct designation（直接指定）]，然後點擊[...]按鈕，在打開的窗口中指定移動的位置。如果想要使用地圖 ID 和坐標指定位置，選擇[Designation with variables（使用變量指定）]，然後選擇要引用的變量的指定[ID]，[X 坐標]，[Y 坐標]。

Set Event Location（設置事件位置）

功能

改變某事件的位置。

設置

Events（事件）
指定目標事件。如果設置為[This Event（本事件）]，那麼事件本身就是要移動的事件。事件的移動僅限制在同一張地圖上（不能將本地圖的事件移動到其他地圖上）。
Location（位置）
指定移動後的位置。如果想要將事件移動到指定位置，選擇[Direct designation（直接指定）]，然後點擊[...]按鈕，在打開的窗口中指定移動的位置。如果想要使用地圖 ID 和坐標指定位置，選擇[Designation with variables（使用變量指定）]，然後選擇要引用的變量的指定[ID]，[X 坐標]，[Y 坐標]。
Direction（方向）
指定事件移動後的朝向。

注記

  • 戰鬥事件中不能使用此指令。

Scroll Map（畫面捲動）

功能

在不改變角色位置的條件下，移動（捲動）地圖的顯示範圍。

設置

Direction（方向）
畫面捲動的方向。
Distance（距離）
指定移動的距離（圖塊的數量）。
Speed（速度）
指定捲動的速度（共 6 個等級）。

注記

  • 戰鬥事件中不能使用此指令。

Set Movement Route（設置移動路線）

功能

強制主角或地圖事件按照指定的路線移動。
詳情請參考[設置移動路線]。

 

注記

  • 設置了移動路線的主角或事件會立即開始移動。
  • 如果角色已經被設置了移動路線，使用本事件指令再次設置移動路線時，之前的移動路線將失效（只有新移動指令會生效）。當你想要在角色完成上一移動路線之後才開始下一路線的移動，你需要選擇[Wait for Completion（等待移動結束）]。
  • 選擇了[Wait for Completion（等待移動結束）]後，在移動期間玩家不能進行任何操作（並行處理的事件除外）。如果此時處於移動路線的角色撞到了障礙物上，移動指令將不會繼續執行，事件就會在這個位置卡住。使用[Skip If Cannot Move（忽略不能移動的場合）]來避免這種情況出現。
  • 當使用事件指令移動主角時，所移動的步數不會算在角色的移動步數里面。
  • 戰鬥事件中不能使用此指令。

Get on/off Vehicle（上/下交通工具）

功能

控制玩家進出交通工具。當玩家在交通工具中時，使用此指令玩家會離開交通工具；當玩家在地面上時，使用此指令玩家會進入交通工具。不需要設置參數。

注記

  • 本事件指令僅在玩家使用[決定鍵]的時候強制主角進出交通工具。如果沒有可用交通工具（沒有可進入的交通工具或者沒有可以著陸的地面），那麼此指令沒有效果。
  • 戰鬥事件中不能使用此指令。
