地圖詳細信息

當你新建一個地圖，或者右鍵點擊一個已有地圖然後選擇「編輯」時，地圖屬性窗口就會打開。在本窗口中，你可以編輯地圖大小，所用的工具，背景音樂，遇敵條件(觸發戰鬥的條件)和遊戲的行為。

屬性詳細信息

常規設置

Name(名字)

地圖的名字。這個屬性只在編輯器中使用(不會影響遊戲)。

Display Name(顯示名稱)

在角色進入地圖時顯示在屏幕上的地圖名。

Tileset(圖塊組)

指定地圖佈局所使用的圖塊組。

Width/Height(寬度/高度)

地圖的尺寸。[Width(寬度)] 和 [Height(高度)] 的設置範圍為 0 ~ 256。當縮小地圖尺寸時，地圖以外的部分將被刪除。

Scroll Type(滾動模式)

決定地圖如何循環滾動。當設置為循環模式時，角色可以從地圖的一邊循環走到另一邊。

No Loop(不循環)
地圖不是循環的。
Loop Vertically(垂直循環)
地圖的上下邊界會連接起來。
Loop Horizontally(水平循環)
地圖的上下邊界會連接起來。
Loop Both(水平/垂直循環)
上下邊界，左右邊界都會連接起來。

Encounter Steps(遇敵步數)

決定主角隨機遇敵的頻率。設定值為主角平均走多少步就會遇敵敵人(範圍為 1 ~ 999，一步的含義就是移動一個圖塊的距離)。如果你想要怪物頻繁出現，請把此數值下調。

Autoplay BGM/ Autoplay BGS(自動播放 BGM/BGS)

啟用時，主角進入該地圖後，BGM(背景音樂)和 BGS(背景聲音)將會自動播放。請分別選擇你想要播放的聲音文件。
如圖地圖的 BGM 和戰鬥 BGM 相同，切換場景時，音樂不會停止。

Specify Battleback(指定戰鬥背景)

啟用時，你可以使用兩種圖像的組合來指定此地圖戰鬥的背景。
未指定時，戰鬥背景會自動和地形一致。對其他地圖來講，戰鬥背景圖也可能是處理過的地圖畫面。

Disable Dashing(禁止加速)

啟用時，玩家不能在這個地圖上加速跑。

Parallax Background(遠景圖)

出現在地圖透明處的圖像。點擊[...]按鈕後，在彈出的窗口中選擇你想使用的文件。

當你啟用[Loop Vertically(垂直循環)] 或 [Loop Horizontally(水平循環)]時，背景也會隨著主角移動方向而捲動。並且，如果你設置的滾動值不是 0 (範圍為 -32 ~ 32)，它也會自動滾動。如果滾動值時正數，它會向左或向上滾動，如果滾動值時負數，它會向右或向下滾動。數值的絕對值越大，滾動速度越快。

如果你啟用[Show in the Editor(在編輯器中顯示)]，那麼你就會在編輯器中看到遠景圖預覽。然而，顯示的內容並不總是和實際遊戲中的相同。

此外，如果你在遠景圖文件名的前面加上"!"，該圖的視差會變為 0。

Note(備註)

在製作遊戲時，你可以隨便在這裡添加備註。這個屬性僅僅在編輯器中有效(不會影響遊戲)。

Encounters(遇敵)

這個列表顯示了所有角色在地圖上移動時隨機遇到的敵群。雙擊空行會打開一個窗口，你可以設定以下內容。當右鍵點擊一個已有的敵群時，你可以在右鍵菜單中做複製或刪除等操作。

Troops(敵群)
定義你想要設置的敵群。
Weight(權重)
設置遇敵時此敵群出現的優先順序(0 ~ 100)。權重越高，當定義多個敵群時，遇到這一組敵群的可能就越大。

敵群的遇敵概率計算取決於此敵群權重的所佔比例。例如，敵群 A 的權重為 9，敵群 B 的權重為 7 ，敵群 C 的權重為 4。這種情況下，遇到敵群 A 的概率為 9/20(總權重為 9+7+4)，也就是 0.45。換句話說，它會有 45% 的幾率出現。相似地，敵群 B 出現幾率為 7/20(35%)，敵群 C 的為 20%。
Range(範圍)
敵群出現的區域。如果遇敵範圍和區域無關，請選擇"Entire Map(整個地圖)"。如果要指定僅在特定區域遇敵，請選擇"Specify by Region ID(由區域 ID 指定)"，並指定區域的 ID(最多指定 3 個區域)。設置區域的方法如下。

設置區域

在地圖的遇敵設置中指定的[Region(區域)]表明在此區域你將遇到何種敵人。一張地圖可以分成至多 255 個區域。

要設置區域，首先在圖塊調色板上選擇 R 標籤，切換到編輯模式。之後，在窗口的左上方選擇區域的 ID(1 ~ 255)，然後在地圖上點擊就可以設置相應圖塊的區域。一個圖塊只能設置一個區域 ID。
