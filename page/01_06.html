基本控制

運行你的遊戲

如何運行本軟件所開發的遊戲取決於取決於遊戲的發佈方式。

使用 Windows
在遊戲文件夾中雙擊文件名為[Game.exe](或[Game])的可執行程序。
使用 Mac OS X
在遊戲文件夾中雙擊文件名為[Game.app](或[Game])的可執行程序。
在 Android/iOS 系統上
參考 轉換為 Android 程序和轉換為 iOS 程序.

* 當遊戲為網頁遊戲時，參考[建立本地服務器]以在本地環境中查看你的遊戲。

遊戲內的菜單控制

標題菜單

標題畫面在你運行遊戲後出現。選擇下面的選項以繼續你的遊戲。

New Game(新遊戲)
從頭開始一個新遊戲。
Continue(繼續)
從舊存檔處開始遊戲。接下來需要選擇存檔。
Options(選項)
調整遊戲全局設置，例如 BGM 的音量。

遊戲時的主菜單[待討論，原文是Menu During Movement，直譯感覺會造成誤解]

當玩家在地圖中移動時，按下菜單鍵(也就是取消按鈕，遊戲手柄除外)就會顯示這個主菜單。你在菜單中可以使用道具來恢復角色的狀態，或者是存儲遊戲進度，這些命令可以在左側窗口中找到。各個命令的詳細信息如下。

Item(物品)
查看/使用隊伍所擁有的物品。要使用一件物品，從列表當中選中它即可(此操作取決於你使用的物品，有些物品選中後還要指定使用的目標才能發揮效果)。
Skill(技能)
查看/使用技能(魔法，等等)。要使用一個技能，從列表當中選中它即可(此操作取決於你使用的技能，有些技能選中後還要指定使用的目標才能發揮效果)。
Equip(裝備)
查看/變更裝備。當選擇一個角色並改變他的裝備時，你會同時選中要換下的裝備和即將穿到角色身上的裝備(或者是選擇空白項來卸下裝備)。
Status(狀態)
查看角色的狀態。你需要選擇你想查看的角色。
Formation(編隊)
更改隊伍成員的順序。
Options(選項)
調整遊戲全局設置，例如 BGM 的音量。
Save(存檔)
保存當前遊戲狀態。從 20 個存檔位置中選一個你希望將遊戲進度存儲的位置。
Game End(結束遊戲)
退出遊戲。從 [To Title(返回標題畫面)] 或 [Cancel(取消)] (亦可使用取消鍵)中選擇一項。

戰鬥菜單

當玩家遇敵時，遊戲會切換到戰鬥畫面中。你可以利用下面的命令進行戰鬥。當所有隊員的 HP 降至 0 時，遊戲結束。

隊伍指令
每一回合都會顯示的指令。選擇[Fight(戰鬥)]以繼續戰鬥，或者是[Escape(逃跑)]來從敵群中撤退出來。但是，如果敵人來攻擊你，即便是你選擇了[Escape(逃跑)]，在本回合也只有敵人才能行動。
角色指令
當你選擇了[Fight(戰鬥)]指令時，你需要為隊伍中每位成員下達指令。主要的行動為[Attack(攻擊)](使用當前裝備的武器攻擊敵人)，[Guard(防禦)](自我防護以減少所受傷害)和[Item(物品)](使用當前所擁有的物品)。可選的行動種類取決於角色的設定。

商店菜單

在商店你可以購買或出售物品，武器，防具等。

購買
當你選擇[Buy(購買)]時，屏幕中會出現可購買的物品，武器，防具等。選擇物品及其數量然後按下 OK 按鈕確認購買。

所持數/可裝備此裝備的角色
隊伍所持該物品的數量會顯示在右側窗口中。

當購買武器和防具時，隊員會顯示在右邊，無法裝備此裝備的成員會顯示為灰色。當你的隊員數在 5 或以上時，你可以利用衝刺鍵或點擊(輕觸)右側窗口來顯示第五名隊員後面的成員。
出售
當你選擇[Sell(出售)]並選擇[Item(物品)]，[Weapon(武器)]，[Armor(防具)]，[Key Item(重要物品)]，你當前擁有的物品會顯示在屏幕中。選擇你想出售的物品並輸入數量，按下 OK 按鈕確認出售。
