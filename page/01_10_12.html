Battle Processing（戰鬥處理）

功能

導致敵群出現，開始一場戰鬥。

設置

Troop（敵群）
指定和玩家發生戰鬥的敵群。選擇「直接指定」來選取固定敵群和角色發生戰鬥，如果想利用 ID 指定敵群，選擇「使用變量指定」，然後輸入要引用的變量。如果選擇「和隨機遇敵相同」，則主角遭遇的敵群會在當前地圖的[Encounter（遭遇）]列表中隨機選擇一個。
Can Escape（允許逃跑）
啟用時，[Escape（逃跑）]指令在戰鬥中可用，同時你也可以指定「勝利的場合」和「逃跑的場合」兩種條件分歧。
Can Lose（允許失敗）
啟用時，主角隊伍被擊敗後遊戲不會結束，同時你也可以指定「勝利的場合」和「失敗的場合兩種條件分歧。

注記

  • 此指令不能用於戰鬥事件。

Shop Processing（商店處理）

功能

呼叫商店畫面，玩家可以在裡面購買武器，防具，物品。

設置

Merchandise（商品）
指定商店中出售的物品。雙擊列表的空行後，你可以在打開的窗口中設置[Merchandise（商品）]和[Price（價格）]。選擇[Standard（標準）]選項後，商品的價格和數據庫中的價格相同，選擇[Specify（指定）]選項後，商品的價格會臨時變更成此事件指令中指定的值。
Purchase Only（僅可購買）
啟用時，玩家不能在商店中出售已有物品。

注記

  • 此指令不能用於戰鬥事件。

Name Input Processing（名稱輸入處理）

功能

顯示角色姓名輸入畫面，玩家可以在此進行輸入，更改角色名字。

設置

Actor（角色）
指定要改名的角色。
Max characters（最大字符數）
指定姓名的最大字符數（1 ~ 16）。

注記

  • 此指令不能用於戰鬥事件。
  • 遊戲中需要輸入姓名時，你可以使用十字鍵移動光標，按下確定鍵輸入字符，按下取消鍵刪除字符。

Open Menu Screen（呼叫菜單畫面）

功能

呼叫菜單畫面。不需要設置參數。

注記

  • 此指令不能用於戰鬥事件。

Open Save Screen（呼叫存檔畫面）

功能

呼叫存檔畫面。不需要設置參數。

注記

  • 此指令不能用於戰鬥事件。

Game Over（遊戲結束）

功能

強制遊戲結束，顯示遊戲結束畫面。不需要設置參數。

Return to Title Screen（返回標題畫面）

功能

強制遊戲返回到標題畫面。不需要設置參數。
