方法

addChild ( child ) Object

往容器中添加一個子對象。

參數：

  • child Object

    要添加的子對象。

返回值：

Object:
被添加的子對象。

addChildAt ( child  index ) Object

將子對像添加到指定索引位置。

參數：

  • child Object

    要添加的子對象。

  • index Number

    欲添加到的位置索引。

返回值：

Object:
被添加的子對象。

move ( x , y , width , height )

同時設置x坐標、y坐標、寬度和高度。

參數：

  • x Number

    窗口層的x坐標。

  • y Number

    窗口層的y坐標。

  • width Number

    窗口層的寬度。

  • height Number

    窗口層的高度。

removeChild ( child ) Object

從容器中移除一個子對象。

參數：

  • child Object

    要移除的子對象。

返回值：

Object:
被移除的子對象。

removeChildAt ( index ) Object

移除指定索引處的子對象。

參數：

  • index Number

    要移除的子對象的索引。

返回值：

Object:
被移除的子對象。

update ()

更新窗口層。

屬性

children Array

[只讀] 窗口層的子對像數組。

height Number

窗口層的高度。

parent Object

[只讀]窗口層的父對象。

width Number

窗口層的高度。

x Number

窗口層的x坐標。

y Number

窗口層的y坐標。
