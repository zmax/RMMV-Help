Plugin Manager（插件管理器）

選擇[Tools（工具）] → [Plugin Manager（插件管理器）]就可以打開[Plugin Manager（插件管理器）]，使用[Plugin Manager（插件管理器）]你就可以管理官方插件和你自製的插件。

當你點擊列表中顯示的一個插件時，會出現一個對話框。在對話框中你可以啟用/禁用此插件或者調節一些參數值。

當輸入參數時，右鍵打開菜單，選擇[IconSet Viewer...]可以顯示圖標查看器。
（2016.03.01 更新）

Sound Test（聲音測試）

選擇[Tools（工具）] → [Sound Test（聲音測試）]你就可以打開[Sound Test（聲音測試）]窗口，在這裡你可以測試工程中的音樂和聲音文件。當你關閉聲音測試窗口時，音樂會繼續播放。因此你可以使用此功能來在製作遊戲的同時播放背景音樂。

窗口被分為四個標籤頁：BGM（背景音樂），BGS（背景聲音），ME（音效）和 SE（聲效）。你可以在各個標籤頁的文件列表中選擇音樂。當你點擊[Play（播放）]按鈕時音樂就會開始播放。點擊[Stop（停止）]按鈕時音樂播放就會停止。

你可以通過旁邊的滑動條來分別調節聲音的 Volume（音量）（0 ~ 100%），Pitch（音調）（50 ~ 150%）和 Pan（聲場定位）（-100 ~ 100）。

Event Searcher（事件查找器）

選擇[Tools（工具）] → [Event Searcher（事件查找器）]你就可以打開[Event Searcher（事件查找器）]，在這裡你可以通過開關，變量，名稱來查找事件，並把它們顯示在列表當中。

Character Generator（角色生成器）

選擇[Tools（工具）] → [Character Generator（角色生成器）]來打開角色生成器。在這裡你可以生成角色的圖形，並把它們分配到主角和事件中去。

點擊[Male（男）]或[Female（女）]標籤來改變性別，並根據你想要的特點來選擇變化和顏色。你的選擇也會在[Preview（預覽）]區域中顯示。

Randomize（隨機）
隨機設置所有部分。
Face Image（頭像）
打開角色頭像的導出窗口。
Walk Character（角色行走圖）
打開角色行走圖的導出窗口。
Damage Character（角色倒地圖）
打開角色倒地圖的導出窗口。
Battler（戰鬥圖）
打開橫版戰鬥圖的導出窗口。
Save Settings（保存設置）
存儲所製作的角色數據。
Load Settings（載入設置）
載入保存的角色數據。

導出窗口

Import
從文件中導入一張待修改的基礎素材圖，並將當前製作的新素材插入其中。
Export
將當前圖像導出為文件。

Generate Dungeon（迷宮生成器）

右鍵點擊地圖列表中的地圖並選擇[Generate Dungeon（迷宮生成器）]即可打開迷宮生成器。

當你選擇牆體和地板圖塊時，地圖會自動被繪製成迷宮的樣子。由於要在所選地圖上生成一個迷宮，因此當你要生成大型迷宮時，你需要增加地圖尺寸。如果地圖尺寸太小，那麼生成出來的地圖會不太像一個迷宮。

Type（種類）
有[Rooms（房間）]和[Maze（迷宮）]兩種風格可選。當你選擇[Rooms（房間）]，迷宮生成器就會生成一個帶有房間和過道的迷宮；當你選擇[Maze（迷宮）]，那麼生成的迷宮只會有過道（也就是所謂真正意義上的迷宮）。
Options（選項）
Add Margins（添加邊緣）
當選擇此選項時，地圖的上、下、左、右會自動添加邊緣。
Wide passages（通道加寬）
當選擇此選項時，迷宮中的過道會變寬。
Wall
選擇填充牆體的圖塊。
* 從正在使用的 A4 (牆體) 圖塊組中選擇。
Floor
選擇填充地板的圖塊。
* 從正在使用的 A5 (地板) 圖塊組中選擇。

Playtest（遊戲測試）

你可以在製作遊戲過程中的任何階段使用[Playtest（遊戲測試）]功能來測試你的遊戲，這項功能可以在菜單欄中的[Game（遊戲）] → [Playtest（遊戲測試）]找到。通過進行遊戲測試，你可以在遊戲發佈之前查看遊戲的背景和事件是否按照預期進行運作。

使用調試功能
當運行遊戲測試時，按下鍵盤上的 F9 會打開調試窗口。在這裡你可以改變屏幕中所顯示的開關和變量的值。
* 在調試界面再按下[F9]即可返回遊戲測試。

要改變變量的值，你需要選擇目標開關/變量，（上方的 S 表示開關，下方的 V 表示變量，旁邊的數字表示編號範圍），然後按下決定鍵，右側的目標開關/變量窗口會出現一個光標。使用決定鍵來改變開關的 ON/OFF 值，使用左右方向鍵和 L/R 來改變變量的值。
如何使用開發者工具
在遊戲測試中按下 F8 鍵可以打開開發者工具。開發者工具的功能基本上與 Google Chrome 的相似。你可以檢查任何使用 HTML5 編寫的遊戲中的元素。
退出遊戲測試
要退出遊戲測試，按下窗口上的 X 按鈕以關閉窗口。

Resource Manager（資源管理器） （2016.03.01 更新）

選擇[Tools（工具）] → [Resource Manager（資源管理器）]即可打開資源管理器。在[Resource Manager（資源管理器）]中, 你能夠管理工程中包含的資源。窗口中各個按鈕的功能將在下面說明。想要在遊戲中使用你自己的原創文件（包括圖片/音頻等），你的素材需要有特定的標準。詳情請參考[素材標準]。

文件夾列表
窗口的左側會顯示包含資源文件的文件夾。當導入資源文件時，你需要根據你使用的目的來選擇目標文件夾。
文件列表
窗口的右側會顯示所選文件夾中的文件列表。
Preview（預覽）
預覽[文件列表]中選中的圖片文件。如果想要檢查聲音文件，請使用[Sound Test（聲音測試）]工具。
Import（導入）
向工程中導入資源文件。首先你要在[文件夾列表]中選擇要導入資源文件的位置，然後再選擇文件進行導入。
Export（導出）
將工程中的資源文件保存到工程之外。導出後，工程中依然含有導出的資源文件。
Delete（刪除）
刪除[文件列表]中的資源文件。文件一旦刪除無法恢復，因此刪除前請仔細考慮。
