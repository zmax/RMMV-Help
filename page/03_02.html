方法

canPlayVideoType ( type ) Boolean [static]

檢測瀏覽器能否播放指定格式的視頻。

參數：

  • type String

    待檢測的視頻格式

返回值：

Boolean:
若瀏覽器可以播放該格式視頻，返回 true 

canUseDifferenceBlend () Boolean [static]

檢測 canvas 是否支持差值混合模式。

返回值：

Boolean:
若支持該混合模式，返回 true 

canUseSaturationBlend () Boolean [static]

檢測 canvas 是否支持飽和度混合模式。

返回值：

Boolean:
若支持該混合模式，返回 true 

endLoading () [static]

清除「正在讀取中」狀態的圖片。

hasWebGL () Boolean [static]

檢測瀏覽器是否支持 WebGL。

返回值：

Boolean:
若瀏覽器支持 WebGL，返回 true 

hideFps () [static]

隱藏 FPS 顯示。

initialize ( width , height , type ) [static]

初始化畫面系統。

參數：

  • width Number

    遊戲畫面的寬度

  • height Number

    遊戲畫面的高度

  • type String

    渲染器的類型（'canvas'、'webgl' 或 'auto'）

isFontLoaded ( name ) Boolean [static]

檢測字體文件是否載入。

參數：

  • name String

    字體的文件名

返回值：

Boolean:
若成功載入，返回 true 

isInsideCanvas ( x , y ) Boolean [static]

檢測指定點是否在遊戲的 canvas 區域內。

參數：

  • x Number

    指定點的 X 坐標

  • y Number

    指定點的 Y 坐標

返回值：

Boolean:
若指定點在 canvas 內，返回 true 

isVideoPlaying () Boolean [static]

檢測是否正在播放視頻。

返回值：

Boolean:
若正在播放視頻，返回 true 

isWebGL () Boolean [static]

檢測渲染器的類型是否為 WebGL 。

返回值：

Boolean:
若渲染器類型為WebGL，返回 true 

loadFont ( name , url ) [static]

讀取字體文件。

參數：

  • name String

    字體文件的文件名

  • url String

    字體文件的地址

pageToCanvasX ( x ) Number [static]

將頁面的 X 坐標轉換為 canvas 中與之對應的 X 坐標。

參數：

  • x Number

    頁面的 X 坐標

返回值：

Number:
canvas 中與之對應的 X 坐標

pageToCanvasY ( y ) Number [static]

將頁面的 Y 坐標轉換為 canvas 中與之對應的 Y 坐標。

參數：

  • y Number

    頁面的 Y 坐標

返回值：

Number:
canvas 中與之對應的 Y 坐標
