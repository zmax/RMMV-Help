  • 使用 Enigma Virtual Box

即使您在遊戲打包時選擇了 Windows 平台並導出了遊戲文件夾，玩家也沒法直接點擊 Game.exe 開始遊戲。你必須將整個遊戲文件夾都發佈出去。

但是，使用 Enigma Virtual Box 打包遊戲目錄後，您可以只發佈一個單獨的 .exe 文件。

下載和安裝

  1. 點擊下方的地址下載 [Enigma Virtual Box] 。
    http://enigmaprotector.com/en/downloads.html
  2. 雙擊下載到的 "enigmavb.exe" 文件，按照提示安裝軟件。

使用方法

  1. 運行 Enigma Virtual Box.
  2. 點擊 "Enter Input File Name:（待轉換文件：）" 輸入框旁邊的 [Browse...（瀏覽...）] 按鈕，選擇之前輸出的 .exe 文件。"Enter Output File Name:（輸出位置：）" 輸入框中的內容將自動設置。（您可以自由的修改文件地址以及文件名。）
  3. 點擊 [Add（添加）] 按鈕，選擇 [Add Folder Recursive（將文件夾添加到隊列）]。
  4. 選擇導出的遊戲目錄，一個新的窗口會在你點擊 [OK（確定）] 後跳出來。直接點擊 [OK（確定）] 即可。

    ↓
  5. 確認剛才選擇的目錄已經顯示在窗口中後（圖中紅框部分）（喵5：哪有 =口= ！），點擊 [Process（開始）] 按鈕。
  6. 一個單獨的 .exe 文件將出現在輸出位置輸入框所設置的文件夾中。
