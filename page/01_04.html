有關主窗口菜單欄中的選項的細節如下。

[File]菜單

New Project
創建新的工程。如果一個工程已經被打開，那麼當創建新工程的同時，這個工程將會被關閉。
Open Project
打開保存的工程，這使得你可以繼續編輯你的遊戲。進入到你的工程目錄，並在[open]窗口中選擇你的[Game](或者[Game.rpgproject])文件。
Close Project
關閉當前打開的工程。如果數據未保存，那麼你將會看到一個確認對話框。點擊[Yes]將保存並關閉工程，點擊[No]將關閉工程，但是不保存。
Save Project
保存你當前正編輯的工程的內容。
Deployment
根據你所選發佈平台建立你的工程。具體請參考[輸出格式]。
Exit RPG Maker MV
關閉程序。和[Close Project]一樣，如果有未儲存的數據，那麼你將會看到一個確認窗口。

[Edit] 菜單

Undo
允許你撤銷剛剛的操作，將工程恢復到一步之前的狀態。你至多可以撤銷20步操作。
Cut
將你選中的內容(例如地圖數據和地圖事件)刪除，並把它們複製到剪貼板上。
Copy
將你選中的內容(例如地圖數據和地圖事件)複製到剪貼板上。
Paste
將剪貼板上的內容(例如地圖數據和地圖事件)添加到所選位置。
Delete
刪除選中的數據(例如地圖數據和地圖事件)。

[Mode] 菜單

Map
切換到地圖設計器模式。
Event
切換到事件編輯模式。在此模式下，地圖上會出現網格，網格的大小和圖塊大小相當。
* 在圖塊窗口選擇[R]標籤可以進入區域編輯模式，在此模式下你可以決定敵人將出現在何區域(遇敵區域)。

[Draw] 菜單

在圖塊上繪製的工具已經可以在地圖編輯模式下使用了。詳情請參考[編輯地圖] 。

[Scale] 菜單

Zoom In
當你每次選擇「放大」時，地圖預覽窗的內容會變得更大。
Zoom Out
當你每次選擇「縮小」時，地圖預覽窗的內容會變得更小。
Actual Size
以標準大小顯示地圖(實際大小和當前編輯器中的大小相同).

[Tool] 菜單

Database
打開[Database]窗口，以便創建/編輯遊戲數據(例如角色和道具)。
Plugin Manager
你在這裡管理你的插件。詳情請參考[如何使用輔助工具]。
Sound Test
你可以播放導入到工程中測試聲音。詳情請參考[如何使用輔助工具]。
Event Searcher
你可以使用各種各樣的條件搜索事件。詳情請參考[如何使用輔助工具]。
Character Generator
將事先做好的素材組合起來，生成角色臉譜，行走圖，戰鬥圖。詳情請參考[如何使用輔助工具]。

[Game] 菜單

Playtest
開始遊戲測試。詳情請參考[如何使用輔助工具]。
Open Folder
打開工程所在的文件夾。當你要查看工程文件夾的位置或者手動將文件移動到工程文件夾中時，請使用這個功能。

[Help] 菜單

Contents
顯示幫助窗口(本窗口)。
About
顯示此軟件的版本信息。
