地圖事件創建和控制

切換到編輯模式

為了創建和編輯地圖事件，首先通過點擊主窗口工具欄中的[事件](Events)按鈕切換到事件編輯模式（菜單欄中使用[Mode]（模式） -> [Events]（事件）也可以切換）

創建一個新的地圖事件

要創建新地圖事件，先在地圖上雙擊你需要放置事件的地方。然後在彈出的[Event Editor](事件編輯器)中編輯你的設定，按下[OK]按鈕確定。要獲得更多關於設置參數的幫助，請查看[地圖事件設置].

再編輯和移動

當你放置一個地圖事件的時候，一個圖標將會在你事件的位置顯示（你第一個事件頁選擇的圖像會默認被使用）。雙擊這個圖像，會再次打開[Event Editor]（事件編輯器）窗口，這裡你可以再次編輯事件設置。此時，你也可以通過拖動圖標來移動你的事件位置。

通過上下文菜單控制

當你在事件編輯模式右擊地圖的時候，你可以複製或者刪除已經放置的事件。上下文菜單有如下功能。

Edit（編輯） (當圖塊沒有事件的時候，這裡是「New」，新建事件)
打開事件編輯器窗口
Cut （剪切）
複製當前事件到剪貼板並刪除當前事件
Copy （複製）
複製你右擊的地圖事件
Paste （粘貼）
粘貼剪貼板的地圖時間到你右擊的位置
Delete （刪除）
刪除你右擊的地圖事件
Quick Event Creation （快速創建事件）
使用默認配置創建四個類型的事件（稍後會提到）
Set Starting Position（設置初始點）
設定玩家和交通工具的起始點。起始點將會被藍色框標識。
*事件圖標使用白色框

快速事件創建

通過使用上下文菜單中的[Quick Event Creation](快速事件創建)，你可以創建四種傳統RPG中常用的事件類型（使用默認設置）：[Transfer]（場所移動）, [Door](門), [Treasure]（寶物）, [Inn]（旅館）

就像其他事件一樣，使用「快速事件創建」創建的事件一樣可以編輯，複製其的參數和進行其他可以對標準事件進行的操作。

Transfer（場所移動）
在地圖上的指定位置創建一個不可見的事件，將玩家傳送到指定位置。請在樓梯或者入口這種地方使用這個事件。在[Location](位置)和[Direction]（方向）參數中指定玩家被傳送的目的地和玩家的朝向。
Door（門）
創建包含一張圖像的事件，它會將玩家傳送到指定位置。使用[Image]（圖像）字段指定顯示的圖像，用[Location]（位置）字段指定玩家被傳送到的位置。
Treasure（寶物）
創建一個可以在玩家第一次打開時從寶箱獲得物品的事件。使用[Image]（圖像）字段指定顯示的圖像，通過[Content]（執行內容）字段指定玩家能獲得什麼樣的物品（金幣/物品/武器/防具）
Inn（旅館）
創建一個可以讓你的隊伍完成回復的事件，使用[Image]（圖像）字段指定顯示的圖像， 使用[Price]（價格）字段指定回復的價格。
